#liquid REST API

REST API  for tierce applications

## Running project

You need to have installed Node.js and MongoDB 

### first Install dependencies 

To install dependencies enter project folder and run following command:
```
npm install
```
configure the .ENV file that fits your needs

start mongoDB service 
create bdd "liquid-db-dev" or rename as you specified on your .ENV file 

### Run server
To run server execute:
```
npm start 
```

done ... :)
